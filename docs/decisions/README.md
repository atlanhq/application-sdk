# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Atlan Application SDK.

## Format

Each ADR follows this structure:

- **Title**: Short descriptive name
- **Status**: Accepted, Deprecated, or Superseded
- **Context**: The forces at play and why a decision was needed
- **Decision**: What we decided and why
- **Consequences**: Trade-offs and implications
- **References**: Links to relevant source files
